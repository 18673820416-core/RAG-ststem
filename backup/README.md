# RAG系统 - 记忆系统化项目

## 项目定位
基于"意识=认知=记忆=意义"循环等式，建立个人知识记忆系统化管理的RAG系统。

## 设计原则
- **简单实用**：避免过度设计，先解决记忆流失问题
- **系统化管理**：将零散记忆转化为结构化知识
- **可扩展性**：为后续检索功能预留接口

## 技术栈
- 向量数据库：SQLite + sqlite-vector扩展
- 文本嵌入：轻量级模型（如all-MiniLM-L6-v2）
- 数据源：PC端对话历史、浏览器缓存、IDE日志

## 项目结构
```
RAG系统/
├── README.md           # 项目说明
├── data/              # 数据存储
├── src/               # 源代码
├── docs/              # 文档
└── config/            # 配置文件
```

## 开发计划
1. 建立基础文件夹结构
2. 实现数据收集模块
3. 构建向量化存储
4. 开发基础检索功能

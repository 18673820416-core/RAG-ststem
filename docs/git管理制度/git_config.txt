## 🚀 常用Git命令速查

### 初始化远程仓库
# 方法1：HTTPS方式（推荐）
git remote add origin https://github.com/你的用户名/rag-system.git

# 方法2：SSH方式
git remote add origin git@github.com:你的用户名/rag-system.git

### 首次推送
git add .
git commit -m "初始提交：RAG系统完整代码"
git branch -M main
git push -u origin main

### 日常操作
git status                    # 查看状态
git add .                     # 添加所有更改
git commit -m "提交信息"      # 提交更改
git push                      # 推送到远程
git pull                      # 拉取远程更新

### 回滚操作
git log --oneline -5          # 查看最近5次提交
git reset --soft HEAD~1       # 软回滚（保留更改）
git reset --hard HEAD~1       # 硬回滚（丢弃更改）
git revert <commit_id>        # 安全回滚（创建新提交）

# 2. 定期更新GitHub Token

## 📞 问题排查
# 如果推送失败，尝试：
git pull origin main --rebase  # 先拉取并变基
git push origin main           # 再推送

# 如果冲突严重，可以强制推送（谨慎使用）
git push origin main --force

# 查看远程分支
git branch -r

# 查看所有分支
git branch -a

# 🔧 Git仓库管理制度

此目录包含项目的Git版本控制管理制度和相关文档。

## 📋 文档清单

### 📖 核心制度文档
- `Git提交规范指南.md` - 详细的提交规范和文件分类标准
- `Git回滚操作指南.md` - 完整的回滚操作指南和安全策略
- `git_config.txt` - 仓库配置信息和常用命令速查

### 🚀 快速开始
1. 阅读`Git提交规范指南.md`了解提交标准
2. 查看`git_config.txt`获取仓库配置信息
3. 需要回滚时参考`Git回滚操作指南.md`

### 📞 问题反馈
如在使用过程中遇到问题，请参考相关文档或联系项目维护人员。

# 🚀 LLM驱动记忆梳理系统

## 📋 系统概述

基于你的建议，我已经成功实现了一个完整的**LLM驱动记忆梳理系统**，使用DEEPSEEK API来对记忆数据进行全面梳理，包括去重、知识图谱构建和语义关联维补全。

## 🎯 核心功能

### 1. **智能记忆梳理** (`src/memory_organizer.py`)
- **去重分析**: 使用LLM进行语义级去重，识别重复或高度相似的记忆
- **语义分析**: 深度分析记忆内容的主题、概念、情感和复杂度
- **知识图谱构建**: 构建增强版知识图谱，优化节点和边的关系
- **网状关联建立**: 自动创建记忆间的语义关联网络

### 2. **DEEPSEEK API集成** (`config/deepseek_config.py`)
- **API配置管理**: 支持DEEPSEEK API密钥的配置和测试
- **智能切换**: 自动检测API密钥，无密钥时使用模拟模式
- **错误处理**: 完善的错误处理和重试机制

### 3. **网状思维引擎集成**
- **记忆存储**: 与现有网状思维引擎无缝集成
- **关联构建**: 自动创建思维节点和关联边
- **增强搜索**: 支持网状增强的语义搜索

## 🔧 使用方法

### 1. **配置DEEPSEEK API密钥**

运行配置向导来设置API密钥：

```bash
python setup_deepseek.py setup
```

或者使用交互式配置：

```bash
python setup_deepseek.py
```

### 2. **运行记忆梳理演示**

查看系统功能演示：

```bash
python demo_memory_organizer.py
```

### 3. **完整测试流程**

运行完整测试：

```bash
python setup_deepseek.py test
```

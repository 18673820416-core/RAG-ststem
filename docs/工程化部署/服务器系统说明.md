## 📋 原始内容（仅供参考）

## 系统概述

RAG系统采用单一服务器架构，所有功能集成在一个稳定的服务器文件中，确保系统简洁可靠。

## 服务器组件

### 当前系统服务器

**启动文件**: `stable_start_server.py`
- **定位**: 集成式多智能体聊天系统
- **端口**: 10808
- **入口页面**: `start.html` (启动页面)
- **聊天页面**: `templates/chatroom.html`
- **功能**: 多智能体聊天室、网状思维引擎、认知破障引擎、智能体管理等
- **特点**: "无门即安全"设计，端口即密码，一键启动

**启动方式**:
```bash
python stable_start_server.py
```

**访问地址**: http://localhost:10808

## 重要说明

1. **系统采用单一服务器架构**，所有功能集成在 `stable_start_server.py` 中
2. **不存在多个服务器同时运行**的情况，避免端口冲突和资源浪费
3. **系统启动简单**：运行 `python stable_start_server.py` 即可启动所有功能
4. **访问便捷**：通过 http://localhost:10808 访问启动页面

## 最佳实践

1. 启动系统时使用 `python stable_start_server.py`
2. 访问入口页面: http://localhost:10808
3. 点击"进入系统"按钮跳转到聊天室页面
4. 所有功能通过单一服务器提供，无需管理多个服务

## 故障排除

- 如果端口冲突，检查是否有其他服务器在运行
- 确保Python环境正确配置
- 检查依赖包是否完整安装
